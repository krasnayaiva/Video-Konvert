# Video Konvert

**Video Konvert** — это удобный инструмент для конвертации видео из одного формата в другой. Простой, быстрый и написанный на Python.

<img width="1366" height="768" alt="изображение" src="https://github.com/user-attachments/assets/923d0660-1ea7-42eb-bf0e-bb8c322efa4d" />

## ✨ Возможности (Демо-версия)

✅ **Конвертация в основные форматы:**
- MP4
- AVI
- MOV
- MKV
- WebM
- 3GP
- FLV

✅ **Базовые настройки:**
- Выбор качества выходного видео
- Сохранение оригинального соотношения сторон

✅ **Простой интерфейс:**
- Интуитивно понятное окно настроек
- Прогресс-бар конвертации
- Окно логов конвертации
- Информация о выходном файле (выбор каталога для сохранения)
